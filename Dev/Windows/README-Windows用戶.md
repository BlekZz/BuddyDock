# BuddyDock for Windows — 安裝與使用說明

> 版本：1.0.3 ｜ 平台：Windows 10 / 11（64 位元）

---

## 下載哪個檔案？

`release/` 資料夾內有兩個 EXE，選一個即可：

| 檔案 | 適合誰 | 說明 |
|---|---|---|
| `BuddyDock-1.0.3-portable.exe` | 想直接試用、不想安裝 | 雙擊即執行，不寫入系統，隨時刪除 |
| `BuddyDock-1.0.3-setup.exe` | 想長期使用 | 安裝到電腦，建立開始選單捷徑 |

---

## 方式一：免安裝（便攜版）

1. 下載 `BuddyDock-1.0.3-portable.exe`
2. 雙擊執行
3. 等待約 5–10 秒解壓（首次執行稍慢，之後正常）
4. 桌面右下角出現動物角色即成功

**注意事項：**
- 便攜版每次執行都會重新解壓到暫存資料夾，這是正常現象
- 你的設定、語錄清單、使用記錄都儲存在 `%AppData%\BuddyDock\`，不會因為刪除 EXE 而遺失

---

## 方式二：安裝版

1. 下載 `BuddyDock-1.0.3-setup.exe`
2. 雙擊執行
3. 安裝程式會**自動靜默安裝**，不需要任何點擊，完成後自動開啟 BuddyDock
4. 之後可從開始選單搜尋「BuddyDock」開啟

**安裝位置：** `%LocalAppData%\Programs\BuddyDock\`  
**使用者資料：** `%AppData%\BuddyDock\`（設定、清單皆儲存於此）

**解除安裝：**  
設定 → 應用程式 → 搜尋「BuddyDock」→ 解除安裝

## Windows 安全警告說明

初次執行時，Windows 可能彈出以下提示，這是正常現象：

### 「Windows 已保護您的電腦」（SmartScreen）

> 此應用程式未經 Microsoft 認證的程式碼簽署。

**處理方式：**
1. 點擊「**更多資訊**」
2. 點擊「**仍要執行**」

這個警告出現的原因是：本程式沒有花費數百美元購買商業程式碼簽署憑證，與安全無關。

### 防毒軟體誤報

部分防毒軟體（如 Windows Defender）可能對未簽署的 EXE 發出警告。如遇此情況，加入例外即可。

## 基本操作

啟動後，動物角色會出現在**螢幕右下角**。

| 操作 | 說明 |
|---|---|
| **左鍵拖曳** 角色 | 移動到任意位置（可半身探出螢幕邊緣） |
| **右鍵點擊** 角色 | 開啟主選單 |
| **系統匣圖示**（右下角工具列） | 左鍵：顯示 / 聚焦；右鍵：快捷選單 |

### 右鍵選單功能

- **切換角色** — 黑貓 / 馬爾濟斯 / 島嶼長尾雀
- **勾選清單** — 快速切換要顯示的語錄清單
- **詳細設定** — 開啟完整設定頁
- **說明書** — 功能說明
- **隱藏** — 暫時隱藏角色（系統匣仍在）
- **關閉** — 完全退出程式

## 設定頁說明（右鍵 → 詳細設定）

設定頁分三個分頁：

### 清單（語錄管理）
- 新增 / 編輯 / 刪除語錄清單
- 點 🤍 收藏語錄，可開啟「只從收藏抽語錄」
- 匯入 / 匯出 `.json` 格式備份

### 互動
- **自動換句**：每 N 分鐘自動換一句語錄
- **定時提醒**：每 N 分鐘跳出系統通知
- **靜音時段**：設定期間不發送通知（例如 22:00 → 08:00）
- **開機自動啟動**：登入 Windows 後自動開啟
- **字型大小**：小 / 中 / 大

### 日程
- 設定特定時間的一次性 / 每日 / 每週提醒

## 常見問題

**Q：角色不見了，怎麼辦？**  
A：點系統匣的 BuddyDock 圖示（右下角工具列），或右鍵 → 顯示。

**Q：通知沒有跳出來？**  
A：確認 Windows 通知設定是否已允許 BuddyDock 發送通知：設定 → 系統 → 通知 → 確認 BuddyDock 已開啟。

**Q：如何備份我的語錄清單？**  
A：設定 → 清單 → 匯出，存成 `.json` 檔案。換電腦時用匯入功能還原。

**Q：視窗透明區域顯示黑色？**  
A：需要 Windows 10 版本 1903 以上。確認顯示卡驅動程式為最新版本。

**Q：開機自動啟動設定後沒有生效？**  
A：若使用便攜版，自動啟動路徑可能因暫存位置變動而失效。建議改用安裝版後再設定開機啟動。
